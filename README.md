# playwright-framework-saucedemo
Web UI automation framework to test Sauce Demo's ecommerce website. E2E, login and edge case testing is built into the framework for as close to 100% test coverage as possible.

## LINKS
Sauce Demo: https://sauce-demo.myshopify.com/

## GOALS
The main goals of this project is to showcase use of Playwright and Typescript automation testing. From E2E to API data-validation testing.
